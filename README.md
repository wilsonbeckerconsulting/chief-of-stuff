# Chief of Stuff

Private command center for managing Shopify (full-time) + two freelance clients.

## Navigate
`SCHEDULE.md` → time off, meetings | `clients/INDEX.md` → who | `clients/<client>/README.md` → client details | `projects/INDEX.md` → active work | `projects/<project>.md` → project details | `TIME.md` → billable hours | `raw/` → brain dumps (search by tags) | `.secrets/` → credentials & API keys (gitignored, local only)

## Structure
- `SCHEDULE.md` - Time off, recurring meetings
- `clients/` - Client context folders
  - `clients/<client>/README.md` - Client info, contacts, context
  - `clients/<client>/dev/` - Dev workspace for exploratory work (gitignored credentials)
- `projects/` - Active work, todos, deadlines
- `TIME.md` - Billable hours for invoicing
- `raw/` - Verbatim brain dumps (tagged, searchable)
- `archive/` - Historical record
  - `archive/accomplishments/` - Weekly logs of completed work (for client comms)
- `.secrets/` - Credentials, API keys (gitignored, local only)
