# Chief of Staff

Private command center for managing Shopify (full-time) + two freelance clients.

## Quick Start
Open in Cursor → new thread → say `initialize`

## Structure
- `SCHEDULE.md` - Time off, recurring meetings
- `clients/` - Client context (who they are, key contacts)
- `projects/` - Active work, todos, deadlines
- `TIME.md` - Billable hours for invoicing
- `raw/` - Verbatim brain dumps (tagged, searchable)

## Commands
- `initialize` - Load context, check date, report ready
- `brain dump` - Capture voice-to-text, extract todos
- `review` - Validate projects, clean up, prioritize
- `log time: [client] [hrs] - [notes]` - Track hours
- `what do I need to do tonight?` - Prioritized tasks
- `help me with [problem]` - Context-aware assistance
